現在制作中のDiscord Botについて

色々機能は作ったのですが、今回はその中で、荒らし対策用に 作成している機能の一つの、特定のキーワードを含むメッセージの 自動削除機能を紹介します。
というのも具体的にどういった機能なのか、例えば「test」という キーワードを含むメッセージを削除したいとき、事前に「test」と いう文字列をbotに登録しておくことによって、送信した時に瞬時に削除 されます(ログは確認用)

→

処理の内容としても、

メッセージが送信される→テキストファイルから登録したキーワードの一覧を取り出す→メッセージに登録したキーワードが含まれていたらメッセージを削除してログを送信する と簡単に作れます。
